# BOOKING FORM

Webhook form for the n8n booking system.

![alt text](image.png)

## Run Locally

Clone the project

```bash
  git clone https://github.com/KeyMailer/n8n-booking-system-forms.git
```

Go to the project directory

```bash
  cd n8n-booking-system-forms
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
